# Internship Application Tracker

## 📌 Project Description
This is a Python-based CRUD project that helps users manage and track their internship applications easily.

## 🚀 Features
- Add internship applications
- View all applications
- Search applications by company
- Update application status
- Delete applications
- View statistics (total, approved, rejected)

## 🛠 Tech Used
- Python

## ▶ How to Run
1. Download or clone this repository
2. Run `main.py` in any Python environment
3. Follow the menu options
